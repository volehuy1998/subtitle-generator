About SubForge

Open-source AI subtitle generation — fast, private, and free to self-host.

What is SubForge?

AI-powered transcription

Upload any audio or video file and receive accurate subtitles in SRT, VTT, or JSON format within seconds to minutes. Powered by OpenAI's Whisper model via the faster-whisper (CTranslate2) runtime.

100% on-premise

Nothing is sent to external APIs. Transcription runs entirely on your server using open-source models. Your media files never leave the machine.

Free and open source

SubForge is MIT-licensed. Inspect every line of code, self-host on your own infrastructure, or contribute improvements back to the project.

Capabilities

Multi-language

Transcribes and translates 90+ languages. Auto-detect or specify language manually.

Subtitle Embedding

Soft-mux generated subtitles directly into your video in one click — no re-upload needed.

GPU Accelerated

Uses NVIDIA CUDA when available for 10–20× faster transcription. Falls back to CPU automatically.

Multiple Formats

Export as SRT (universal), WebVTT (web), or JSON (developers). All formats include word-level timestamps.

Private by Design

Files deleted within 24 hours. No accounts. No telemetry. Sessions are anonymous.

Real-time Progress

Live progress updates via Server-Sent Events. See exactly which step the pipeline is on.
